# Site de Anúncios - HTML5, CSS3, JavaScript, php e JSON

## Descrição
Este projeto é um site de anúncios simples desenvolvido com HTML5, CSS3, JavaScript, php e JSON. Permite que um administrador cadastre novos anunciantes e seus anúncios, que são exibidos na 
página principal.

## Tecnologias Utilizadas
* **HTML5:** Estrutura da página.
* **CSS3:** Estilização da página.
* **JavaScript:** Lógica do site, incluindo o cadastro de anúncios e a exibição dos mesmos.
*  **PHP** Gerenciamento do user admin.
* **JSON:** Armazenamento dos dados dos anúncios (pode ser substituído por um banco de dados).
